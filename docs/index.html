Simulasi Model Ising 2D

Tugas Fisika Komputasi - Kelompok Barcelona

Proyek ini mengimplementasikan Model Ising 2 Dimensi menggunakan Algoritma Metropolis (Monte Carlo) untuk mempelajari fenomena transisi fasa, magnetisasi spontan, dan perubahan perilaku sistem pada berbagai kondisi suhu.

Kasus 1 - Fase Feromagnetik (T = 1.0) Kasus 2 - Transisi Fasa (T = 2.27) Kasus 3 - Fase Paramagnetik (T = 4.0)
