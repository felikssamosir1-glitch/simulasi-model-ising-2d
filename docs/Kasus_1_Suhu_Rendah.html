Kasus 1: Fase Feromagnetik (T = 1.0)

Kernel: Python 3 Simulasi Monte Carlo Kelompok Barcelona

Tujuan

Mensimulasikan Model Ising 2 Dimensi menggunakan algoritma Metropolis pada suhu rendah (T = 1.0) untuk mengamati perilaku sistem ketika energi internal mendominasi agitasi termal, sehingga orientasi spin menyelaraskan diri secara spontan (Fase Feromagnetik)[cite: 1].

Parameter Simulasi

  • Ukuran kisi : 20 × 20[cite: 1]
  • Suhu (T) : 1.0[cite: 1]
  • Monte Carlo Steps : 200.000 Langkah[cite: 1]
  • Inisialisasi : Hot Start (Konfigurasi Acak Murni)[cite: 1]

Kode Python

In [1]: Algoritma Metropolis Python 
import numpy as np
import matplotlib.pyplot as plt

# ==========================================
# Parameter Simulasi
# ==========================================
N = 20                 # Ukuran kisi N x N[cite: 1]
T = 1.0                # Suhu sistem (T < Tc, dengan Tc ≈ 2.269)[cite: 1]
MC_STEPS = 200000      # Total langkah Monte Carlo[cite: 1]
RECORD_INTERVAL = 100  # Interval pengambilan sampel makrostate[cite: 1]

# ==========================================
# Inisialisasi Sistem
# ==========================================
# Hot Start: Keadaan awal acak untuk merepresentasikan entropi maksimum (T -> tak hingga)[cite: 1]
grid = np.random.choice([-1, 1], size=(N, N))[cite: 1]

# Array penampung data
magnetization_history = [][cite: 1]
time_steps = [][cite: 1]

def calculate_magnetization(config):
    """Menghitung magnetisasi rata-rata sistem"""[cite: 1]
    return np.sum(config) / (N * N)[cite: 1]

# Catat keadaan awal
magnetization_history.append(calculate_magnetization(grid))[cite: 1]
time_steps.append(0)[cite: 1]

# Pra-kalkulasi probabilitas Boltzmann (Eksponensial) untuk optimasi kecepatan[cite: 1]
# Perubahan energi (dE) pada kisi persegi hanya memiliki 5 kemungkinan nilai: -8, -4, 0, 4, 8 (dengan J=1)[cite: 1]
exp_dE = {dE: np.exp(-dE / T) for dE in [-8, -4, 0, 4, 8]}[cite: 1]

# ==========================================
# Loop Algoritma Metropolis-Hastings
# ==========================================
for step in range(1, MC_STEPS + 1):[cite: 1]
    # 1. Pilih koordinat spin secara acak[cite: 1]
    i, j = np.random.randint(0, N, size=2)[cite: 1]

    # 2. Terapkan Syarat Batas Periodik (Periodic Boundary Conditions / PBC)[cite: 1]
    spin = grid[i, j][cite: 1]
    neighbors = (grid[(i + 1) % N, j] +
                 grid[(i - 1) % N, j] +
                 grid[i, (j + 1) % N] +
                 grid[i, (j - 1) % N])[cite: 1]

    # 3. Hitung perubahan energi jika spin ini dibalik (dE = 2 * S_i * Sum(S_j))[cite: 1]
    dE = 2 * spin * neighbors[cite: 1]

    # 4. Kriteria Penerimaan Metropolis[cite: 1]
    # Jika energi turun (dE <= 0) ATAU probabilitas termal memenuhi syarat[cite: 1]
    if dE <= 0 or np.random.rand() < exp_dE[dE]:[cite: 1]
        grid[i, j] *= -1  # Balik orientasi spin[cite: 1]

    # 5. Sampling data makroskopik[cite: 1]
    if step % RECORD_INTERVAL == 0:[cite: 1]
        magnetization_history.append(calculate_magnetization(grid))[cite: 1]
        time_steps.append(step)[cite: 1]

# ==========================================
# Visualisasi Data
# ==========================================
fig, axes = plt.subplots(1, 2, figsize=(14, 6))[cite: 1]

# Plot 1: Keadaan Akhir Kisi (Domain Magnetik)[cite: 1]
im = axes[0].imshow(grid, cmap='binary', vmin=-1, vmax=1)[cite: 1]
axes[0].set_title(f'Keadaan Akhir Kisi Spin\n(T = {T}, N = {N})', fontsize=14)[cite: 1]
axes[0].axis('off')[cite: 1]

# Plot 2: Riwayat Magnetisasi[cite: 1]
axes[1].plot(time_steps, magnetization_history, color='black', linewidth=1.5)[cite: 1]
axes[1].set_title('Riwayat Magnetisasi Rata-Rata', fontsize=14)[cite: 1]
axes[1].set_xlabel('Langkah Monte Carlo', fontsize=12)[cite: 1]
axes[1].set_ylabel('Magnetisasi ($M$)', fontsize=12)[cite: 1]
axes[1].set_ylim(-1.1, 1.1)[cite: 1]
axes[1].grid(True, linestyle='--', alpha=0.7)[cite: 1]

plt.tight_layout()[cite: 1]
plt.show()[cite: 1]

Hasil Analisis

Pada suhu T = 1.0, sistem berada di bawah suhu kritis ($T_c \approx 2.27$), sehingga pengaruh interaksi antar-spin lebih kuat dibandingkan energi termal[cite: 1]. Hal ini membuat spin cenderung tersusun searah dan membentuk domain besar yang seragam[cite: 1]. Magnetisasi rata-rata bernilai jauh dari nol, menandakan adanya magnetisasi spontan[cite: 1]. Kondisi tersebut menunjukkan bahwa sistem berada dalam fase feromagnetik, yaitu fase ketika sebagian besar spin teratur dengan arah yang sama[cite: 1].
