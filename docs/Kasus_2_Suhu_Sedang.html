Kasus 2: Transisi Fasa dan Kritikalitas (T = 2.27)

Kernel: Python 3 Simulasi Monte Carlo Kelompok Barcelona

Tujuan

Mensimulasikan Model Ising 2 Dimensi menggunakan algoritma Metropolis pada suhu kritis (T = 2.27) untuk mengamati terjadinya transisi fasa, fluktuasi kritis, dan perubahan perilaku magnetisasi sistem secara makroskopis.

Parameter Simulasi

  • Ukuran kisi : 20 × 20
  • Suhu (T) : 2.27 (Suhu Kritis Teoritis Onsager)
  • Monte Carlo Steps : 200.000 Langkah
  • Inisialisasi : Hot Start (Konfigurasi Acak Murni)

Kode Python

In [29]: Algoritma Metropolis Python 
import numpy as np
import matplotlib.pyplot as plt

# ==========================================
# Parameter Simulasi
# ==========================================
N = 20                 # Ukuran kisi N x N
T = 2.27               # Suhu Kritis Teoritis (T ≈ Tc)
MC_STEPS = 200000      # Total langkah Monte Carlo
RECORD_INTERVAL = 100  # Interval pengambilan sampel makrostate

# ==========================================
# Inisialisasi Sistem
# ==========================================
# Hot Start: Konfigurasi acak untuk merepresentasikan entropi maksimum
grid = np.random.choice([-1, 1], size=(N, N))

# Array penampung data makrostate
magnetization_history = []
time_steps = []

def calculate_magnetization(config):
    """Menghitung nilai rata-rata magnetisasi kisi"""
    return np.sum(config) / (N * N)

# Catat kondisi awal sebelum loop
magnetization_history.append(calculate_magnetization(grid))
time_steps.append(0)

# Pra-kalkulasi faktor Boltzmann untuk menghemat komputasi di dalam loop
exp_dE = {dE: np.exp(-dE / T) for dE in [-8, -4, 0, 4, 8]}

# ==========================================
# Loop Algoritma Metropolis-Hastings
# ==========================================
for step in range(1, MC_STEPS + 1):
    # 1. Pilih spin secara acak di dalam kisi
    i, j = np.random.randint(0, N, size=2)

    # 2. Ambil nilai spin saat ini dan tetangga terdekat dengan Syarat Batas Periodik (PBC)
    spin = grid[i, j]
    neighbors = (grid[(i + 1) % N, j] +
                 grid[(i - 1) % N, j] +
                 grid[i, (j + 1) % N] +
                 grid[i, (j - 1) % N])

    # 3. Hitung perubahan energi potensial (dE) jika spin dibalik
    dE = 2 * spin * neighbors

    # 4. Evaluasi Probabilitas Metropolis
    if dE <= 0 or np.random.rand() < exp_dE[dE]:
        grid[i, j] *= -1  # Terima pembalikan spin

    # 5. Pencatatan Berkala Makrostate
    if step % RECORD_INTERVAL == 0:
        magnetization_history.append(calculate_magnetization(grid))
        time_steps.append(step)

Pada suhu T = 2.27, sistem berada di sekitar suhu kritis (Tc), yaitu batas antara fase feromagnetik dan paramagnetik. Pada kondisi termodinamika ini, energi interaksi antar-spin yang berusaha mensejajarkan spin saling bersaing kuat dengan agitasi energi termal yang memicu ketidakteraturan.

Persaingan ini menghasilkan fluktuasi kritis yang sangat besar secara makroskopis. Domain-domain magnetik tunggal mulai terpecah menjadi kelompok-kelompok kluster dengan ukuran yang bervariasi (membentuk geometri fraktal yang self-similar). Magnetisasi rata-rata sistem ($M$) berosilasi dan berubah secara tidak stabil sepanjang waktu simulasi. Fenomena ini merupakan ciri khas kuat terjadinya Transisi Fasa Orde Kedua dan menandakan runtuhnya keteraturan jarak jauh (long-range order) pada Model Ising 2 Dimensi.
